Automatic File Type Icons

When you include links to non-html pages, small icons are automatically placed next to the file name warning the user that a helper app is needed to read the file. For example:

  • My PDF document
  • My Word Document
  • My Excel Document
  • Email links

However for esign forms, if you wish to have an icon you will need to add the

<span class="esign">My Esign</span>

to the item which then looks like this:

  • My Esign
